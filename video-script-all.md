## EDITING GUIDE FOR CAPCUT / DAVINCI

### Typography Captions — Style Rules
```
Font:          Montserrat ExtraBold or Syne Bold (download free)
Size:          Large — fill 70% of screen width for key lines
Colour:        White with 2px black outline (standard)
Position:      Lower third (bottom 25% of screen)
Animation:     Word-by-word pop (CapCut auto-caption style)

SPECIAL LINES — override style:
  "Ravi loses ₹640"         → Red text, larger size
  "We predict them"          → Purple text, hold 1.5s
  "₹600 credited to UPI"    → Teal/green text, larger
  "Ravi filed nothing"       → White, slow fade, centred mid-screen
  "12 million workers"       → Bold white, massive size
```

### Cuts and Transitions
```
Person → Screen recording:   Hard cut (no transition)
Screen → Screen:             Hard cut
Segment → Segment:           0.1s dip to black (subtle)
Final card:                  Fade to black → logo
```

### Music
- Search "lo-fi corporate background music no copyright" on YouTube Audio Library
- Volume: 15–20% under voice
- Fade out at 1:55 for the silent "Ravi filed nothing" moment — this lands harder in silence

### CapCut Workflow (Mobile)
1. Import all clips in order
2. Auto-caption → manually edit key lines for style
3. Add text overlays for the 6 special lines listed above
4. Add background music track
5. Export 1080p

### For Maximum Views (LinkedIn / YouTube)
- Add subtitles in first 3 seconds — people scroll with sound off
- First frame: show ₹640 loss stat or "12 million workers" — hook immediately
- Thumbnail: Ravi's story line or the ₹600 payout confirmation screen
- Description: paste the problem statement + team name + hashtags
